## 使用说明
### 初次使用
1. 删除 `鬼谷八荒\Mods` 目录下的 `Hylas.dll`（不要删除 `Hylas` 文件夹）
2. 将 `YamlDotNet.dll` 放入 `鬼谷八荒\MelonLoader\Managed` 目录
3. 将 `Butian.dll` 放入 `鬼谷八荒\Mods` 目录
4. 启动鬼谷八荒（初次运行自动从 `Hylas` 导入）
